# Awesome Auto-GPT

Auto-GPT is a pioneering open-source project designed to demonstrate the power of the GPT-4 language model. Harnessing the impressive abilities of GPT-4, this application seamlessly links together Large Language Model (LLM) "thoughts" to independently accomplish any objective you designate. As one of the premier instances of GPT-4 operating in a completely autonomous manner, Auto-GPT is redefining the frontier of AI capabilities.

## Research Papers

- [Auto-GPT for Online Decision Making: Benchmarks and Additional Opinions algorithm](https://arxiv.org/abs/2306.02224)
- This paper presents a comprehensive benchmark study of Auto-GPT styled agents in decision-making tasks. It compares the performance of popular large language models (LLMs) such as GPT-4, GPT-3.5, Claude, and Vicuna. Additionally, it introduces the Additional Opinions algorithm, an effective method to incorporate supervised/imitation-based learners into the Auto-GPT scheme.
- [Lottery and Sprint: Generate a Board Game with Design Sprint methodology](https://arxiv.org/abs/2307.00348)
- This paper introduces the use of the Auto GPT system alongside Design Sprint methodology to facilitate board game creation for inexperienced users. It discusses the implementation of Auto GPT for generating diverse board games and the subsequent optimization process. A user study is conducted to evaluate the playability and enjoyment of the generated games.
- [AutoML-GPT: Automatic Machine Learning with GPT](https://arxiv.org/abs/2305.02499)
- This paper proposes the AutoML-GPT, which utilizes GPT as a bridge to automate the training pipeline of diverse AI models. It dynamically takes user requests from the model and data cards to compose corresponding prompts, conducting experiments from data processing to hyperparameter tuning.
- [AD-AutoGPT: An Autonomous GPT for Alzheimer's Disease](https://arxiv.org/abs/2306.10095)
- Inspired by AutoGPT, this study develops a tool called AD-AutoGPT for autonomous data collection, processing, and analysis of complex health narratives related to Alzheimer's Disease. It leverages the GPT-4 large language model and provides insights into public focus on Alzheimer's Disease.
- [From GPT to AutoGPT: a Brief Attention in NLP Processing using DL](https://www.researchgate.net/publication/370107237_From_GPT_to_AutoGPT_a_Brief_Attention_in_NLP_Processing_using_DL)
- This paper presents the evolution of "Generative Pre-Training (GPT) models" from GPT-1 to the new AutoGPT. It explores the advancements in natural language processing and the capabilities of AutoGPT models.
- [arXiVeri: Automatic table verification with GPT](https://arxiv.org/abs/2306.07968)
- This paper proposes the task of automatic table verification (AutoTV) to verify the accuracy of numerical data in tables by cross-referencing cited sources. It introduces the arXiVeri benchmark dataset and evaluates the performance of a table verifier using modern large language models (LLMs).
- [The Potential Breakthroughs with ChatGPT in Parasitology](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC10423911/)
- This article discusses ChatGPT, an AI model based on generative pretrained transformers (GPT), developed by OpenAI. It highlights the potential breakthroughs and applications of ChatGPT in the field of parasitology.
- [One step of Auto-GPT with Additional Opinions](https://www.researchgate.net/figure/One-step-of-Auto-GPT-with-Additional-Opinions-Here-Additional-Opinions-are-generated-by_fig1_371311588): Auto-GPT is an autonomous agent that leverages recent advancements in adapting Large Language Models (LLMs) for decision-making tasks.
- [Reference-free and cost-effective automated cell type annotation](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC10153208/): GPT-4, a highly potent large language model, can automatically and accurately annotate cell types.
- [HuggingGPT: Solving AI Tasks with ChatGPT and its Friends](https://arxiv.org/pdf/2303.17580): Language serves as an interface for LLMs (e.g., ChatGPT) to connect numerous AI models for solving tasks.
- [Towards Building the Federated GPT: Federated Instruction Tuning](https://arxiv.org/abs/2305.05644): Introduces Federated Instruction Tuning (FedIT), leveraging federated learning for instruction tuning of LLMs.
- [AugGPT: Leveraging ChatGPT for Text Data](https://arxiv.org/abs/2302.13007): Proposed AugGPT approach for text data augmentation, improving performance in few-shot learning text classification tasks.
- [Many bioinformatics programming tasks can be automated](https://arxiv.org/abs/2303.13528): Evaluates the extent to which ChatGPT can successfully complete basic- to moderate-level programming tasks in the field of bioinformatics.
- [ChatGPT Asks, BLIP-2 Answers: Automatic Questioning Towards](https://arxiv.org/abs/2303.06594): Introduces ChatCaptioner, an automatic questioning method deployed in image captioning, where ChatGPT asks informative questions to BLIP-2, a vision question-answering model.
